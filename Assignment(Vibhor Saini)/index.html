Q.4 Difference btw Var , Let and const

Var

1.var: Var has function scope. If a variable is declared inside a function using var, it is only accessible within that function.
2.var: Hoists the variable declaration to the top of its scope, but the initialization remains in its original position. This means you can use the variable before it is declared without causing an error.
3.var allow reassignment. You can change the value of a variable declared with var.
4.When declared outside any function, becomes a property of the global object. (e.g., window in a browser environment).

let and const

1.let and const: Have block scope. They are limited to the block in which they are defined, such as loops or if statements.
2.let and const: Also hoist, but unlike var, they are not initialized until the interpreter reaches the actual declaration in the code. Trying to use the variable before the declaration results in a ReferenceError.
3. let allow reassignment. You can change the value of a variable declared with let. but const Does not allow reassignment after initialization. Once a value is assigned to a const variable, it cannot be changed.
4. let and const: Do not become properties of the global object when declared outside any function.
